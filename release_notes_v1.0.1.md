# v1.0.1

## 应用管理器增强
- 新增 APK 安装功能，可在应用管理器中直接选择本地 APK 安装到当前设备
- 新增分包安装功能，支持一次选择多个 APK 并通过 `adb install-multiple -r` 安装
- 新增 ADB 命令执行页，可直接输入并执行 ADB 子命令
- 增加 ADB 历史命令记录与复用能力
- 内置常用 ADB 命令模板，支持快速填入与执行
- 支持 `{pkg}` / `{device}` 占位符，便于针对当前应用和当前设备快速发命令
